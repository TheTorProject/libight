# NAME
templates &mdash; OONI test templates

# LIBRARY
MeasurementKit (libmeasurement_kit, -lmeasurement_kit).

# SYNOPSIS
```C++
#include <measurement_kit/ooni.hpp>
```

# STABILITY

2 - Stable

# DESCRIPTION

This submodule contains OONI test templates. They are currently undocumented.
