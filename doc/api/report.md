# NAME
report -- Tests results management library

# LIBRARY
MeasurementKit (libmeasurement_kit, -lmeasurement_kit).

# SYNOPSIS
```C++
#include <measurement_kit/report.hpp>
```

1 - Experimental

The `report` library contains code to manage tests results.
